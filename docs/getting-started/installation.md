# Installation

Learn how to install Lista Recurring Events on your WordPress site.

## Download the Plugin

1. Purchase the plugin from [listapage.au](https://listapage.au/custom-wordpress-plugins/recurring-events/)
2. Once you are logged into your account, you can download Lista Recurring Events plugin from the Downloads tab.

![Plugin Download](/img/account-page-download.jpg)

## Install via WordPress Admin

1. Log in to your WordPress admin dashboard
2. Navigate to **Plugins → Add New**
3. Click the **Upload Plugin** button at the top of the page
4. Click **Choose File** and select the downloaded `.zip` file
5. Click **Install Now**
6. After installation completes, click **Activate Plugin**

![Plugin Installation](/img/upload-plugin.jpg)

## Install via FTP

If you prefer manual installation:

1. Extract the `lista-recurring-events.zip` file on your computer
2. Connect to your server via FTP
3. Upload the `lista-recurring-events` folder to `/wp-content/plugins/`
4. Log in to WordPress admin and navigate to **Plugins**
5. Find "Lista Recurring Events" and click **Activate**

## Verify Installation

After activation, you should see:

- A new **Recurring Events** menu item in your WordPress admin sidebar
- A "Recurrence Settings" metabox when editing event posts

![Recurring Events Menu](/img/verify-activation.jpg)

## Activate Your License

To unlock Pro features:

1. Go to **Recurring Events → Settings**
2. Click the **License** tab
3. Enter your license key
4. Click **Activate License**

![License Activation](/img/license-activation.jpg)

:::tip
Your license key was emailed to you after purchase and can be found in your account. Check your spam folder if you can't find it.
:::
